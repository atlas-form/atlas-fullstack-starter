# Where To Put Files

## 目录说明

- `user_docs/`：给你阅读的说明文档
- `requirements/`：你放需求文档
- `development_docs/backend/`：AI 写后端开发文档
- `development_docs/frontend/`：AI 写前端开发文档
- `api_docs/`：AI 写接口文档
- `frontend/`：前端代码
- `backend/`：后端代码

## 你最常接触的目录

通常你只需要关心这几个目录：

1. `user_docs/`
2. `requirements/`
3. `development_docs/`
4. `api_docs/`
